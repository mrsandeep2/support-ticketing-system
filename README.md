# Ticketing System

Full-stack Ticketing System built for IT support / customer service use cases with role-based access control.

## Live Demo

- Application: https://support-ticketing-system-omega.vercel.app/

---

## Tech Stack

| Layer | Tech |
|---|---|
| Frontend | Next.js 14, TypeScript, Tailwind CSS |
| Backend | Spring Boot 3.2, Java 17 |
| Auth | JWT + BCrypt |
| Database | PostgreSQL |
| Deploy | Vercel + Render |

## Features

- Role-based Authentication (Admin, Agent, User)
- JWT Login & Authorization
- Ticket Creation & Management
- Ticket Assignment
- Ticket Status Tracking
- Comments & Ticket History
- Admin Dashboard
- Agent Dashboard
- User Dashboard
- Ticket Priority Management
- File Attachments
